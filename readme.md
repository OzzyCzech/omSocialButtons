# omSocialButtons

Add Twitter, Facebook, Flatter and Google Plus to all Wordpress posts

## How to install

Download Zip version of [master](https://bitbucket.org/OzzyCzech/omsocialbuttons/get/master.zip) and unzip to 
`wp-content/plugins/` directory.
